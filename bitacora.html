Bitácora del Proyecto

Decisiones de Diseño

Paleta de Colores

Decidimos usar una paleta moderna con tonos azules y grises para transmitir profesionalismo y confianza. El color principal (#2563eb) se complementa con grises neutros para crear un diseño limpio y legible.

Tipografía

Seleccionamos Poppins de Google Fonts por su excelente legibilidad y aspecto moderno. Esta fuente funciona bien tanto en títulos como en texto corrido, manteniendo la consistencia visual.

Layout y Estructura

Optamos por un diseño de grid responsivo que se adapta a diferentes tamaños de pantalla. La navegación es simple y directa, priorizando la experiencia del usuario.

Dificultades Encontradas

Responsive Design

Problema: Inicialmente tuvimos dificultades para que el grid de miembros se viera bien en dispositivos móviles.

Solución: Implementamos media queries específicas para cada breakpoint (400px, 900px, 1200px) y ajustamos el número de columnas del grid según el tamaño de pantalla.

Metodología BEM

Problema: Al principio fue desafiante mantener la consistencia en la nomenclatura BEM entre todos los miembros del equipo.

Solución: Establecimos una guía de nomenclatura clara y realizamos revisiones de código para asegurar la consistencia.

Interactividad con JavaScript

Problema: Coordinar las funciones JavaScript entre diferentes páginas sin repetir código.

Solución: Creamos archivos JS modulares (main.js para la página principal y profile.js para las páginas individuales).

Cambios Importantes

Estructura de Archivos

Inicialmente teníamos todos los estilos en un solo archivo CSS. Decidimos mantener un solo archivo pero organizarlo mejor con comentarios y secciones claras para facilitar el mantenimiento.

Navegación

Agregamos botones de "Volver al equipo" en cada página individual para mejorar la navegación, cumpliendo con el requerimiento de no depender del botón "Atrás" del navegador.

Interactividad

Implementamos diferentes tipos de interacciones en cada página individual para mostrar variedad: toggle de visibilidad, cambio de colores, animaciones y alertas.
